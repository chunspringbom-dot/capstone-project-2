# 🥛 Milk Nutrition Analysis App

## 📌 Overview
This project is a simple web application that helps users analyze milk products based on their nutrition.

Instead of just showing information, the app helps users:
- Calculate total nutrition based on intake
- Compare with daily recommended values
- Get recommendations based on personal goals

---

## 🎯 Features

### 1. Nutrition Calculation
- Based on 100ml standard values
- Calculates total intake (e.g., 500ml)

### 2. Daily Intake Comparison
- Shows how much of daily nutrients are consumed
- Helps users understand overconsumption

### 3. Recommendation System
Based on user goals:
- Diet (low calorie, low fat)
- Fitness (high protein)
- Lactose intolerance (lactose-free)

The app provides **reasons**, not just scores.
